# Configuration Recommendations

## DNS Provider
- NextDNS (custom profile)
- Encrypted DNS enabled (DoH/DoT)

## Suggested NextDNS Settings
- Enable native Ads & Trackers blocklists
- Enable Affiliate & Tracking Links protection
- Enable Disguised Trackers
- Disable unnecessary analytics (Crashlytics, Firebase telemetry if possible)

## Allowlist Strategy
- Keep Google Play Services functional
- Allow essential domains for:
  - Messaging apps (WhatsApp, Signal)
  - App updates
  - Push notifications

## VPN Recommendations
Recommended providers:
- Mullvad
- IVPN
- Proton VPN

Criteria:
- No-logs policy
- Independent audits
- WireGuard support
- No invasive mobile telemetry
